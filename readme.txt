Creamos un repo en git hub, cuando le demos el nombre y te indique los comandos hay que hacer los siguiente:

---> git add .
---> git commit -m "mensaje"
  se linkea el proyecto al repo...
---> git remote add origin git@github.com:asgoDev/node_json_server_db.git
  se suben los cambios...
---> git push -u origin main
